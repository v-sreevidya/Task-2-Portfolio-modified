Projects

ELECTRIC PIANO USING IC555 can be used to produce sound signals similar to that of a piano. This circuit works on the principal that resistors connected in series with capacitor changes the frequency.

POWER GENERATION USING FOOTSTEPS is a project that demonstrates the viability and promise of power generation using footsteps and piezoelectric sensor materials as a renewable and eco-friendly energy source.

A PASSWORD-BASED DOOR UNLOCKING SYSTEM utilizing the 8051 microcontroller employs a keypad for user input, an LCD display for feedback, and an electric lock for door control. Upon entering a predefined password via the keypad, the microcontroller validates the input and triggers the electric lock to unlock the door if the password matches.
